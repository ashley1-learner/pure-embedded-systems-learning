What is a program?
  - A program is a series of instructions that causes a computer or microcontroller to perform a particular task.
  - A computer program includes more than just instructions. It also contains data and various memory addresses on which the instructions work to perform specific tasks.
Programming languages in Embedded Systems

  - Assembly
